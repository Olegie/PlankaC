# History / Geschichte

Plankalkuel was designed by Konrad Zuse in the 1940s. Historical descriptions
connect the work with Zuse's 1942/43 logic-machine draft and the later 1945
Plankalkuel document. The language was intended as a general notation for
programs rather than machine-specific assembly code. Its original written form
was two-dimensional: the expression line was followed by rows for variable
indices and structure information.

The complete historical language is larger than this project. It contains
typed structures, procedure calls, guarded program equations, loops, sets,
lists, and logical operations. PlankaC keeps a small executable profile around
the calculator domain, then adds compact structure examples: indexed values,
fields, lists, loops, assertions, and chess-style predicates.

## English Summary

PlankaC is a historically oriented source project for a small executable
linear Plankalkuel profile. PlankaMath is the bundled calculator/GUI example.

## Deutsche Zusammenfassung

PlankaC ist ein historisch orientiertes Quelltextprojekt fuer ein kleines
ausfuehrbares lineares Plankalkuel-Profil. PlankaMath ist das mitgelieferte
Taschenrechner- und GUI-Beispiel.
