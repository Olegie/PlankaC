# Source Context / Quellenkontext

Plankalkuel was designed by Konrad Zuse in the 1940s. Published descriptions
connect the work with Zuse's 1942/43 logic-machine draft and the later 1945
Plankalkuel document. The language was intended as a general notation for
programs rather than machine-specific assembly code. Its original written form
was two-dimensional: the expression line was followed by rows for variable
indices and structure information.

The complete language model is larger than this project. It contains typed
structures, procedure calls, guarded program equations, loops, sets, lists,
and logical operations. PlankaC implements a compact executable profile around
the calculator domain and extends it with indexed values, fields, lists,
loops, assertions, relation helpers, typed value families, chess-domain
structures, and backend targets.

## English Summary

PlankaC is a source-oriented implementation project for a small executable
linear Plankalkuel profile. PlankaMath is the bundled calculator/GUI host.

## Deutsche Zusammenfassung

PlankaC ist ein quellenorientiertes Implementierungsprojekt fuer ein kleines
ausfuehrbares lineares Plankalkuel-Profil. PlankaMath ist der mitgelieferte
Taschenrechner- und GUI-Host.
