# Source Context / Quellenkontext

Plankalkuel was designed by Konrad Zuse in the 1940s. Published descriptions
connect the work with Zuse's 1942/43 logic-machine draft and the later 1945
Plankalkuel document. The language was intended as a general notation for
programs rather than machine-specific assembly code. Its original written form
was two-dimensional: the expression line was followed by rows for variable
indices and structure information.

The language model contains typed structures, procedure calls, guarded program
equations, loops, sets, lists, and logical operations. PlankaC implements an
executable profile with indexed values, fields, lists, loops, assertions,
relation helpers, typed value families, PAGE documents, tagged value storage,
board/game chess procedures, and backend targets.

## English Summary

PlankaC is a source-oriented implementation project for an executable linear
Plankalkuel profile. PlankaMath is the bundled calculator/GUI host.

## Deutsche Zusammenfassung

PlankaC ist ein quellenorientiertes Implementierungsprojekt fuer ein
ausfuehrbares lineares Plankalkuel-Profil. PlankaMath ist der mitgelieferte
Taschenrechner- und GUI-Host.
